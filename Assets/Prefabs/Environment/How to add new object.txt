Для начала создать префаб с этим объектом.

На корневой объект префаба должен быть накинут скрипт ObjectPrefabInfo (для обычных объектов) либо унаследованный от него (для объектов с особыми правилами генерации)

Имя объекто важно для оригинального ObjectPrefabInfo, для особых объектов может быть не важно

У ObjectPrefabInfo можно устанавливать ограничения на кол-во, в частности, делать обязательные объекты или устанавливать максимум. Так же можно менять вероятность появления. По умолчанию у всех 1, но можно поставить больше, чтобы объект появлялся чаще, или 0, чтобы реже.
